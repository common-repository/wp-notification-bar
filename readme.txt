== Description ==
Show notifications to relevant users instantly to increase conversions, promote a blog post or, a new product.

= Benefits = 
<ul>
<li>Quick and Easy setup</li>
<li>Custom message & call to action button</li>
<li>Completely customizable - color, text etc</li>
<li>Language independent</li>
<li>Can be made sticky at top of float as user scrolls down</li>
<li>Analytic (available on UsersDelight.com for free)</li>
<li>Extremely slick looking :-)</li>
</ul>

= How it works = 
On setup, Plugin sends an AJAX request to <a href="https://www.usersdelight.com">UsersDelight</a> to save settings (saved locally on wordpress as well) and get a free unique API key. Then, an embedded JS file loads the settings with the given API key 
and shows notification bar as you configured it.

If an email is provided in your settings, you can login to UsersDelight for free so that you can look at in depth analytic on user interaction with notification bar.

== Installation ==

1. Upload the `notification-bar` folder to the `/wp-content/plugins/` directory
2. Activate the WP Notification Bar through the 'Plugins' menu in WordPress
3. Configure the plugin by going to the `Notifications` menu that appears in your admin menu (left)
